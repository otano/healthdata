# healthdata

## exploration du set de donnée :
https://www.kaggle.com/datasets/sonalshinde123/social-media-mental-health-indicators-dataset

Cet ensemble de données contient 5 000 lignes et 15 colonnes, et recueille des informations détaillées sur les habitudes d'utilisation des réseaux sociaux, les modes de vie et l'état de santé mentale correspondant des individus.

Il inclut des attributs démographiques, le comportement relatif au temps d'écran, les types d'interaction, la durée du sommeil, les niveaux d'activité physique, ainsi qu'une classification étiquetée de l'état mental (mental_state).
